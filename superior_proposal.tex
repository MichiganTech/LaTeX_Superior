\subsection*{Resources}
\addcontentsline{toc}{subsection}{Resources}

A curve fit for the data in Figure \ref{FIGURE_02} results in the following 
expressions, indicating linear scaling exhibited by our software suite: 
\begin{equation}
t \:=\: \left( \frac{63.06}{\mbox{NPROC}} \right) \:\times\: 
        \exp\:\left[0.2333\: \times \: N\right]
\label{equation1}
\end{equation}

Using these equations, one can estimate the time required for simulations 
using a finer grid ($N = 48$) and larger number of processors ($NPROC = 8$). 
A general outline of various simulations in our project is as follows:

\begin{enumerate}
  \item 2 sets of 24 runs each.
  \item In a given set, the first run has to complete before a subsequent
        run can be started.
  \item Each run will use 8 CPU cores and last about 240 hours.
  \item Workflow is CPU and memory intensive, and involves data in the order
        of 20 GB. Any temporary/scratch files and folders created will be
        automatically deleted at the end of each run. Also, the software
        suite automatically checkpoints at set intervals during a given run.
\end{enumerate}
